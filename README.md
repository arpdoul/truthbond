# TruthBond 🔍

AI-powered fake news bounty market built on GenLayer.

## What it does
Users submit news article URLs as claims. GenLayer's AI validators fetch the article live, analyze it with LLM reasoning, and reach consensus via Optimistic Democracy on whether it's FAKE or REAL. Truth-tellers earn bounties.

## Deployment
- **Network:** GenLayer Testnet Bradbury (Phase 1)
- **Contract:** 0x21F6D24E5b422780e6253A0F25620AC56246...
- **Explorer:** https://explorer-bradbury.genlayer.com/address/0x21F6D24E5b422780e6253A0F25620AC56246...

## GenLayer Features Used
- Live web access (gl.get_webpage)
- LLM reasoning (gl.eq_principle_prompt_comparative)
- Optimistic Democracy consensus
- Python Intelligent Contracts

## How to interact
Submit a claim via genlayer-js SDK or GenLayer Studio.
